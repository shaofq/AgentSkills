# 通用内容分析技能

## 强制规则（必须遵守）

1. **绝对禁止要求用户提供文件或文本** - 输入中已经包含了所有需要的信息
2. **直接从输入中提取信息** - 输入的"对话历史"部分包含了之前识别的文件内容
3. **立即给出答案** - 不要询问，直接回答用户的问题

## 输入格式说明

你收到的输入格式如下：
```
以下是之前的对话历史，请参考上下文理解用户需求：

用户: [之前的用户消息]
助手: [之前的助手回复，包含OCR识别结果等内容]

---

当前用户需求: [用户当前的问题]
```

## 执行流程

1. 阅读"对话历史"部分，找到之前识别的内容（如发票信息）
2. 根据"当前用户需求"，从历史内容中提取相关信息
3. 直接回答用户的问题

## 示例

**输入**：
```
以下是之前的对话历史：
用户: 识别这个发票
助手: 发票号码：25927000000145916782，金额：¥415.00...

当前用户需求: 发票号是多少？
```

**正确输出**：
根据之前识别的发票内容，发票号是：**25927000000145916782**

**错误输出**（绝对禁止）：
请提供包含发票的文件...
